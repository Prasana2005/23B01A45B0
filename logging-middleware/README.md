# Logging Middleware

This task involved integrating with the provided Logging API.

## Completed Steps

- Registered successfully using the Registration API.
- Generated `clientID` and `clientSecret`.
- Authenticated using the Auth API.
- Obtained a Bearer access token.
- Successfully sent log entries using the Logging API.

The logging functionality developed during this task was reused in the backend assignments (Vehicle Scheduler and Notification System) for recording application events.
